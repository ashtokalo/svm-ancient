       Полная документация по программе BINPAS - BIN to PAS Converter.
                                  Версия 1.1

                                  Оглавление:
  1. Описание программы и её назначениея.
  2. Описание дополнительного модуля BINPASU - Bin to PAS Converter Utility.
  3. Список ключей программы BINPAS.
  4. Список ключей модуля BINPASU.
  5. Пример использования программы BINPAS и модуля BINPASU.
  6. Список файлов которые содержатся в программе BINPAS.


  1. Программа  BINPAS предназначена для создания  файлов в формате паскаля,
  которые бы содержали какие либо двоичные данные из любых файлов, это может
  понадобиться когда в процессе создания программ вам понадобятся какие-либо
  файлы интегрировать в исполняемый файл при этом не потеряв возможность при
  необходимости создать исходный файл без каких либо проблем.
     Также в следую версию  войдут дополнительные команды для более  полного
  и удобного доступа к подключенным файлам.
     В данной версии пока что вы можете только создавать файлы.
  2. Модуль BINPASU предназначен для  согласования работы всех  подключенных
  файлов. Для этого этот модуль надо подключать перед всеми модулями создан-
  ными  программой BINPAS, а в тексте основной программы  которая использует
  эти модули вызвать порцедуру инициализации InitBPU. Эта процедура настроит
  все подключенные файлы и создаст  их список, благодаря чему к файлам можно
  обращатьс по их полным именам. Так же процедура проверяет состав командной
  строки и если в ней есть её ключи (опции) она делает требуемое от неё. Так
  же в зависимости от опции она завершает работу программы. Если в командной
  строке она не обнаружила свои ключи она ничего не делает.
  3. Список ключей и их описание (BINPAS):
     /D   - Каждый файл может иметь описание (строку длинной 40 символов),
            по умолчанию  описания нет, использую  эту опцию можно  задать
            строку описания,  которая в процессе создания  PAS файла будет
            использованна по назначению;
     /PAS - Опция позволяет задать  произвольное имя файла, который  будет
            создан (имеется в виду имя PAS файла), по умолчанию  имя файла
            берется исходное, а расширение меняется на PAS.
     /BPH - Так как к  программе PINPAS были  подключенны файлы  созданные
            ней же, она использует модуль BINPASU, поэтому  можно получить
            помощь от модуля используя эту команду.
  4. Список ключей и их описание (BINPASU):
     /S   - Показать список файлов  которые содержатся в  файле к которому
            был подключен модуль. Кроме того команды выводит ещё и  размер
            файлов и их  описани, общее  количество файлов  подключенных к
            программе  и их общий  размер. По  окончанию  вывода завершает
            работу программы.
     /E   - Вытягивает из программы файл или файлы. Следующим обязательным
            параметром должно быть имя файла который вы хотите вытащить из
            программы. Если в качестве имени файла дать строку *.* , будут
            извлечены все файлы которые содержатся в программе, также если
            в  качестве имени дать  строку *.расширени , где  вместо слова
            ~расширение~ дать какое-либо расширение  все файлы с этим рас-
            ширением будут извлечены из программы.
     /BPH - Вывести помощь по испоьзованию модуля.
     /NBP - Если этот ключ присутствует в командной строке, то все  другие
            ключи для BINPASU будут недействительными, т.е. модуль BINPASU
            не будет реагировать на свои ключи. Это сделано для того, что-
            бы в программе использующей  модуль BINPASU можно было исполь-
            зовать такиеже ключи.
  5. В качестве примера  использования программы  BINPAS и модуля  BINPASU
  я создал  пример - BPDEMO.PAS,  BPDEMO1.BIN, BPDEMO2.BIN,  BPDEMO.BAT. В
  этом примере файлы с расширением BIN содержат строки ~Demo file for test
  ing BINPAS and BINPASU.~  Файл с раширение PAS содержит текст программы,
  которая  подключает  модули BINPASU,  BPDEMO1, BPDEMO2 и  инициализирует
  модуль BINPAS. Файл BPDEMO.BAT содержит строки запуска програмыы  BPDEMO
  с различными ключами.
  6. Список файлов которые содержатся в программе BINPAS.
     BPDEMO.PAS    - Демонстрационная программа
     BPDEMO1.BIN   - её файл
     BPDEMO2.BIN   - её файл
     BPDEMO.BAT    - Файл запускащий демонстрационную программу
     BINPASU5.TPU  - Модуль BINPASU для Turbo Pascal 5.0
     BINPASU6.TPU  - Модуль BINPASU для Turbo Pascal 6.0
     BINPASU7.TPU  - Модуль BINPASU для Turbo Pascal 7.0
     README.TXT    - Документация - этот файл
     BPSOURCE.RAR  - Исходный текст программы BINPAS и модуля BINPASU
                     версия 1.1, файл защищен паролем.
